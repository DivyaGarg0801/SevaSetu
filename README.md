# SevaSetu: Public Services Complaint System

SevaSetu is a full-stack web application that allows citizens to report public service issues and enables authorities to manage and resolve them efficiently.

---

## 🚀 Tech Stack

### 🔹 Option 1: Node.js (Full-Stack - MERN Style)
- Backend: Node.js, Express
- Database: MongoDB
- Frontend: Vite (React)

### 🔹 Option 2: Python (Flask-Based)
- Backend: Flask (Python)
- Database: MySQL
- Frontend: HTML, CSS, JS (Templates)

---

## ⚙️ Prerequisites

### For Node.js Version:
- Node.js installed
- MongoDB running locally

### For Python Version:
- Python 3.x
- MySQL Server
